    -ms-filter:grayscale(1);
    -o-filter:grayscale(1);

页面顶部阴影

body:before{
    content:"";
    position:fixed;
    top:-10px;left:0;width:100%;height:10px;
    box-shadow:0px 0px 0px rgba(0,0,0,.8)
    z-index:100
}

使用属性选择器用于空链接

a[href^="http"]:empty::before{
    content:attr(href);
}

禁用鼠标事件

.disabled{
    pointer-events:none;
}

CSS3新增的pointer-events能够禁用元素的鼠标事件。(IE9+)
