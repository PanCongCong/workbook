Transition 过渡

简介

Transition描述了css状态变化所需要的时间，以及延迟时间、变化速度模式等

img{
    transition: 1s height;
}

注意事项

1） IE10+，目前都支持无前缀使用

2） 大部分css属性都支持transition，参考

3） transition是根据开始状态和结束状态的数值，计算出中间状态。故若其中一个非明确值，则无过渡效果。如height:auto等

transition-timing-function

状态变化速度，默认是ease
1）linear：匀速
2）ease：慢快慢
3）ease-in：加速
4）ease-out：减速
5）ease-in-out：慢快慢
6）step：直接在开始或者结束
7）cubic-bezier：自定义速度模式，参考

写法

完整写法：   transition: 1s 2s height ease

单独定义：
transition-property: height;
transition-duration: 1s;
transition-delay: 2s;
transition-timing-function: ease;

Animation 动画

简介

animation需要指定一个动画的持续时间，以及动画名称
transition描述了css属性状态改变时的过渡；而animation直接定义了一段动画。

div{
    animation:1s rainbow;
}
@keyframes rainbow{
    0% { background:red }
    50% { background:yellow }
    100%{ background:green }
}

animation-iteration-count 播放次数

默认情况下，动画只会播放一次。加入infinite关键字可以无限次播放

div:hover{
    animation:2s rainbow infinite;
}

也可以指定播放次数：

div:hover{
    animation:1s rainbow 2;
}

animation-fill-mode 结束状态

默认情况下，动画结束后会立刻跳回结束状态，通过animation-fill-mode可以改变停留的行为，有以下几个可选项：

1）none。默认值，回到动画开始的状态
2）backwards。让动画回到第一帧的状态。
3）forwads。让动画停留在最终状态。
4）both。根据animation-direction轮流应用forwards和backwards规则。

动画循环播放时，每次都是从结束状态跳回到开始状态，通过此属性可以改变这一行为
